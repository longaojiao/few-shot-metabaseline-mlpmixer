# few-shot-metabaseline-mlpmixer

pytorch 1.8

8GB gpu required

需要将部分文件夹更改为你的文件夹。
直接运行train.py就能出结果。

环境直接点运行，缺啥安装啥。

所有函数都在一个文件上。
